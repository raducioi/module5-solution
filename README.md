# module5-solution

Module 5 과제 솔루션 저장소입니다.

## 개요

이 저장소는 모듈 5 과제에 대한 솔루션 코드를 포함합니다.

## 시작하기

### 요구 사항

- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- 로컬 개발 환경 (선택 사항)

### 설치

저장소를 클론합니다:

```bash
git clone https://github.com/raducioi/module5-solution.git
cd module5-solution
```

### 실행

`index.html` 파일을 브라우저에서 직접 열거나, 로컬 서버를 사용해 실행합니다:

```bash
# Python 3 사용 시
python -m http.server 8000

# Node.js 사용 시
npx serve
```

브라우저에서 `http://localhost:8000` 으로 접속합니다.
